# Keeper - Simple React Note-Taking App

Keeper is a lightweight note-taking application built with React. It allows users to add, view, and delete notes quickly and efficiently.

## Features
- Add new notes with a title and content
- View all created notes in a clean card layout
- Delete notes you no longer need

## Technologies Used
- React (with Hooks like `useState`)
- JavaScript (ES6+)
- CSS for styling (custom styles)

## Getting Started

### Prerequisites
- Node.js (version 14 or above)
- npm or yarn package manager
